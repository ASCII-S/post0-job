在这个文件夹中，你需要帮助我创建**应届生面试大纲**,可以参考网页搜索。
目标是：针对某个面试主题（例如「操作系统」「深度学习」「高性能计算」等），系统化地设计一份可用于全面考核面试者的知识体系大纲。

要求如下：

1. **结构层次**：大纲必须具有清晰的层级结构，依次为：

   * 主题
   * 章节（一级）
   * 小节（二级，如 1.1、1.2）
   * (可以添加更低一级的层级，如三级、四级等)
   * 知识点
   * 问题点/简答题（最细粒度，用于后续拆分为独立文档）

2. **系统性**：

   * 整体要体现一个完整的知识体系结构，而非零散的题目集合。
   * 各章节之间逻辑连贯，能体现知识的递进与覆盖。

3. **内容充分**：

   * 每个章节应覆盖核心概念、关键技术原理、常见考点与综合性问题。
   * 可以参考互联网已有面试题，但必须在此基础上进行系统化整理与补充。

4. **问题点格式**：

   * 最细层级为具体的“问题点/简答题”。
   * 示例：

     ```
     主题：操作系统  
     章节1：进程管理  
       小节1.1：进程与线程  
         子小节1.1.1：线程
            **线程的实现方式**
               用户级线程与内核级线程的区别
               线程上下文切换的开销有哪些 
     ```
